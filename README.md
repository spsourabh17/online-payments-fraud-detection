## 📌 Overview

This project is a **Machine Learning-powered fraud detection system** designed to identify suspicious online transactions in real time.

It integrates a trained ML model with a **Flask web application**, enabling:

* Real-time fraud prediction
* Risk-based classification
* Interactive user input handling

---

## 🧠 Problem Statement

With the rapid adoption of digital payments, fraud detection has become increasingly complex. Traditional rule-based systems are no longer sufficient to detect evolving fraud patterns.

This system solves the problem by:

* Using machine learning for adaptive detection
* Providing instant predictions
* Reducing false positives through data-driven insights

---

## 🚀 Core Features

### 🔮 Intelligent Fraud Prediction

* Real-time transaction analysis
* Probability-based classification
* Risk levels (Low / Medium / High)

### ⚠️ Risk Detection Logic

Identifies suspicious activities such as:

* Unusual transaction patterns
* Abnormal transaction amounts
* Inconsistent account balances
* High-frequency transactions

### 💡 Decision Engine

* ✅ Legit Transaction
* ⚠️ Medium Risk
* 🚨 Fraud Detected

### 🎨 User Interface

* Clean and structured layout
* Simple and user-friendly forms
* Fast response system

## 🖥️ Running the Project

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/spsourabh17/online-payments-fraud-detection.git
cd online-payments-fraud-detection
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the Application

```bash
python backend/app.py
```

Open in browser:

```
http://127.0.0.1:5000
```
